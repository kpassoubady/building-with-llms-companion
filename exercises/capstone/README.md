# Capstone Project: Fleet Service Management

Self-paced project consolidating skills from Chapters 1-8.
Fleet and automotive services scenario with 3 exercises of increasing complexity.

## Exercises

| Exercise | File | Complexity | Description |
|:---------|:-----|:-----------|:------------|
| Issue Classifier | `exercise1_issue_classifier.py` | Low-mid | Classify fleet issues into categories using structured prompts |
| Work Order Parser | `exercise2_work_order_parser.py` | Mid | Parse messy work order text into structured JSON data |
| Fleet Service Advisor | `exercise3_fleet_service_advisor.py` | High | End-to-end advisory system with evaluation harness |

Solutions are in `../../solutions/capstone/`.
